  id: string;

  published_at: string | null;

  category?: string;     // e.g., "Product Updates", "Engineering", "Release"

Deliverable 1: ChangelogList Component (#11) — HIGH
File to create: src/blog/ChangelogList.tsx Also update: src/blog/index.ts (add export), src/index.ts (add export)

What It Does
A timeline-style changelog page component that fetches blog posts filtered by changelog-relevant categories/tags and displays them in a clean timeline format.

Requirements
Data fetching:

Query content_posts table for published posts
Filter by category OR tags that indicate changelog content (e.g., category in ["Release", "Product Updates", "Changelog"] or tags containing "release", "update", "feature", "fix")
Order by published_at descending
Use getBrowserClient() from ../lib/supabase (same pattern as BlogList)
Filter UI:

Filter buttons at top: All, Releases, Updates, Features, Fixes
Clicking a filter shows only posts with the matching tag
"All" shows everything
Timeline Layout:

Group posts by month/year (e.g., "March 2026", "February 2026")
Each month gets a date header
Posts within a month shown as timeline entries with a vertical line connecting them
Each Entry Shows:

Tag badge (colored: Release = blue, Update = green, Feature = purple, Fix = orange)
Date (formatted, e.g., "Mar 4, 2026")
Title (as a link to /blog/${slug} for the full post)
Excerpt or first ~150 chars of content as summary
Clean, minimal design using Tailwind CSS v3
Empty State:

When no changelog posts exist, show a friendly message: "No changelog entries yet. Check back soon for updates."
Use a centered layout with a muted icon (e.g., a scroll or document icon if lucide-react is available, or just text)
Component Signature:

export function ChangelogList(): JSX.Element
No props needed — it fetches its own data (same pattern as BlogList)
Must have "use client" directive at top
Reference Pattern
Follow the same structure as BlogList.tsx:

useState for posts, loading, selected filter
useEffect for data fetching with Supabase query
Skeleton loading state while fetching
Graceful fallback if Supabase client is null
Tag Badge Colors
